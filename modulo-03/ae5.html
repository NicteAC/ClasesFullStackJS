Sistema de Grillas Bootstrap

Estructura base

  • .container o .container-fluid - Contenedor principal
  • .row - Fila que agrupa columnas
  • .col-* - Columnas (sistema de 12 unidades)

Breakpoints responsivos

col-sm-*

≥ 576px

col-md-*

≥ 768px

col-lg-*

≥ 992px

col-xl-*

≥ 1200px

Clases Utilitarias Bootstrap

Espaciado

  • m-* - Margin
  • p-* - Padding
  • mt-*, mb-* - Top/Bottom
  • mx-*, my-* - Horizontal/Vertical

Flexbox

  • d-flex - Display flex
  • justify-content-*
  • align-items-*

Texto

  • text-center
  • text-primary
  • font-weight-bold

Visibilidad

  • d-none - Ocultar
  • d-md-block - Mostrar en MD+

Colores

  • bg-primary, bg-light
  • text-success, text-danger

Personalización con SASS

Bootstrap permite modificar variables SASS para personalizar colores, tipografías y más.

// Variables personalizadas
$primary: #4CAF50;
$font-family-base: 'Roboto', sans-serif;
$border-radius: 0.5rem;

// Importar Bootstrap
@import "bootstrap";

Esto permite mantener la funcionalidad de Bootstrap con tu identidad visual única.

Buenas Prácticas con Bootstrap

✓ Hacer

  • Usar el sistema de grillas correctamente
  • Aprovechar clases utilitarias
  • Personalizar con variables SASS
  • Mantener semántica HTML
  • Importar solo lo necesario

✗ Evitar

  • Sobrescribir clases base directamente
  • Ignorar la responsividad
  • Cargar todo Bootstrap sin usar
  • Sobrecarga de clases en un elemento
  • Modificar archivos core de Bootstrap
