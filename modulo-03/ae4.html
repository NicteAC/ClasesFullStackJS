01

Modelo de Cajas

La base de todo diseño web

¿Qué es el Modelo de Cajas?

Cada elemento HTML es una caja rectangular compuesta por cuatro áreas:

Contenido

Content

Texto, imagen, etc.

Padding

Espacio interno

Border

Contorno visible

Margin

Espacio externo

Propiedades Fundamentales

.elemento {
  /* Dimensiones */
  width: 300px;
  height: 200px;

  /* Espaciado interno */
  padding: 20px; /* todos los lados */
  padding: 10px 20px; /* vertical horizontal */

  /* Borde */
  border: 2px solid #2D5BFF;

  /* Espaciado externo */
  margin: 15px auto; /* centrado horizontal */
}

Box-sizing: El Truco Clave

content-box (por defecto)

width = solo contenido

width: 200px;
padding: 20px;
border: 5px;
/* Ancho real: 250px */

border-box ✓ Recomendado

width = contenido + padding + border

box-sizing: border-box;
width: 200px;
padding: 20px;
/* Ancho real: 200px */
Tip profesional: Siempre usa * { box-sizing: border-box; } al inicio de tu CSS.

Tipos de Display

La propiedad display define cómo se comporta cada caja:

block:
Ocupa todo el ancho disponible
inline: Solo el ancho de su contenido en línea
inline-block: Combina ambos comportamientos

Block vs Inline vs Inline-block

Característica Block Inline Inline-block
Ocupa todo el ancho ✓ Sí ✗ No ✗ No
Nueva línea ✓ Sí ✗ No ✗ No
Acepta width/height ✓ Sí ✗ No ✓ Sí
Margin vertical ✓ Sí ✗ Colapsa ✓ Sí
Ejemplos div, p, h1 span, a, strong img, button

Flexbox: Layout Moderno

Sistema unidimensional para distribuir espacio y alinear elementos

.contenedor {
  display: flex;
  justify-content: center; /* eje principal */
  align-items: center; /* eje cruzado */
  gap: 1rem; /* espacio entre items */
}

Media Queries: Diseño Responsivo

/* Mobile first */
.container {
  width: 100%;
  padding: 1rem;
}

/* Tablet: 768px+ */
@media (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* Desktop: 1024px+ */
@media (min-width: 1024px) {
  .container {
    width: 960px;
  }
}

Buenas Prácticas

✓ Hacer

  • • Usar box-sizing: border-box global
  • • Preferir unidades relativas (rem, %)
  • • Diseñar mobile first
  • • Usar Flexbox/Grid para layouts
  • • Probar en múltiples dispositivos
  • • Mantener sistema de espaciado consistente

✗ Evitar

  • • Tamaños fijos en px para todo
  • • Floats para layout (obsoleto)
  • • Position absolute excesivo
  • • Ignorar el colapso de márgenes
  • • No usar reset CSS
  • • Olvidar probar en móviles

Problemas Comunes y Soluciones

❌ Colapso de márgenes

Márgenes verticales se fusionan

Solución: Usar padding o display: flex

❌ Elemento se desborda

Contenido sale de su caja

Solución: overflow: hidden/auto

❌ Ancho inesperado

Caja más grande de lo esperado

Solución: box-sizing: border-box

❌ Centrado vertical

No funciona con margin: auto

Solución: Flexbox con align-items
