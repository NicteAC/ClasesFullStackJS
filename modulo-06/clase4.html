Carrito de compras interactivo

⏱ Tiempo disponible
45:00
Listo para iniciar

📋 Contexto

Catálogo de 4 productos con nombre y precio. El usuario agrega productos al carrito, ajusta la cantidad con botones + / − y ve el total actualizado en tiempo real.

Consigna:

  1. En data() declarar array productos con { id, nombre, precio } y array carrito vacío
  2. @click en cada producto para ejecutar agregar(producto) — no agregar duplicados
  3. Lista del carrito con v-for; botones + y − con @click cambian item.cantidad (mínimo 1)
  4. Botón eliminar ítem con @click.stop
  5. computed: { total() } que suma precio × cantidad de todos los ítems
  6. Componente ProductoCard.vue con props: ['producto'] que hace this.$emit('agregar', producto)
  7. El padre escucha @agregar="agregarAlCarrito"
  8. Botón "Vaciar carrito" con @click.once — solo funciona una vez

Formulario con $emit y Vuex

⏱ Tiempo de demo
25:00
Listo para iniciar

🛠 Lo que construimos

ContactoForm.vue valida y emite al padre. El padre guarda los mensajes en Vuex y los muestra con un computed que lee el store.
  1. data() en el hijo — email, mensaje, errorMsg, enviando
  2. @submit.prevent en el form — frenar el comportamiento nativo
  3. Método enviar() — validar, luego this.$emit('enviar', { email, mensaje })
  4. @keydown.ctrl.enter en el textarea — atajo de teclado
  5. :disabled="enviando" en el botón mientras procesa
  6. Vuex store — state.mensajes y mutation AGREGAR_MENSAJE
  7. El padre escucha @enviar y llama a this.$store.commit('AGREGAR_MENSAJE', payload)
  8. computed: { mensajes() { return this.$store.state.mensajes } }

Sistema de reseñas con estrellas

⏱ Tiempo disponible
45:00
Listo para iniciar

📋 Contexto

Componente hijo ResenaForm.vue donde el usuario ingresa nombre, selecciona puntuación con estrellas y escribe un comentario. El padre guarda las reseñas en Vuex y muestra el promedio.

Consigna:

  1. data() en el hijo: nombre, puntuacion, puntPreview, comentario, errores
  2. 5 <span> como estrellas: @click asigna puntuacion, @mouseover asigna puntPreview para efecto visual
  3. Validar en methods: nombre no vacío, puntuación 1–5, comentario ≥ 15 caracteres
  4. this.$emit('nueva-resena', { nombre, puntuacion, comentario }) solo si pasa la validación
  5. Vuex store con state.resenas y mutation AGREGAR_RESENA
  6. El padre escucha @nueva-resena y hace this.$store.commit('AGREGAR_RESENA', payload)
  7. Botón "👍 Me gusta" por reseña con @click.stop que incrementa un contador en data()
  8. computed: { promedio() } en el padre que calcula el promedio de puntuaciones del store
