📦 ¿Qué es un Objeto en JavaScript?

Un objeto es una estructura de datos que permite almacenar información en forma de propiedades y métodos.

Propiedades

Pares clave-valor que representan características del objeto.

Métodos

Funciones almacenadas dentro del objeto que permiten ejecutar acciones.

Diferencias con tipos primitivos

  • Los objetos pueden almacenar múltiples valores organizados
  • Son más flexibles: se pueden agregar, modificar o eliminar propiedades dinámicamente

🔑 Propiedades y Métodos de un Objeto

Propiedades

Pares clave-valor dentro de un objeto.

El valor puede ser: número, string, booleano, otro objeto, array, etc.

Ejemplo: nombre, edad, profesion

Métodos

Funciones dentro de un objeto.

Permiten que el objeto realice acciones.

Ejemplo: saludar(), calcular(), mostrar()

⚡ Creación y Uso de Objetos

Forma más común

Literales de objeto usando llaves { }

✅ Recomendado

Simple, directa y legible

Alternativa

Constructor Object()

⚠️ Menos común

Más verbosa y raramente usada

🎯 Notación de Punto y de Corchetes

Notación de Punto

Ejemplo: persona.nombre

Ventaja: Más legible y fácil de escribir

Uso: Cuando el nombre es una cadena válida

Notación de Corchetes

Ejemplo: persona["nombre"]

Ventaja: Acceso dinámico

Uso: Con espacios, caracteres especiales o variables

📊 Cuándo Usar Cada Notación

Situación Notación de Punto Notación de Corchetes
Propiedad con nombre simple ✓ Sí ✓ Sí
Propiedad con espacios o caracteres especiales ✗ No ✓ Sí
Acceso dinámico con variable ✗ No ✓ Sí

🔤 String Primitivo vs Objeto String

String Primitivo

Tipo básico de texto

Ejemplo: "hola"

  • Más eficiente
  • ✅ Recomendado usar

Objeto String

Envoltorio del texto

Ejemplo: new String("hola")

  • Menos eficiente
  • ⚠️ Evitar usar

✨ ¿Qué es "Autoboxing"?

El motor de JavaScript envuelve temporalmente un primitivo para poder usar métodos

¿Cómo funciona?

  • JavaScript envuelve el primitivo en un objeto temporal
  • Ejecuta el método solicitado
  • Descarta el envoltorio automáticamente
  • Retorna el resultado

Regla rápida

  • Usa literales: 'texto', "texto", `texto`
  • Compara con ===
  • Evita new String(...)
