Variables en JavaScript

¿Qué es una variable?

Un espacio en memoria que almacena un valor y puede cambiar durante la ejecución del programa.

let

Para valores que pueden cambiar

let edad = 25;

const

Para valores constantes

const PI = 3.1416;

Tipos de Datos

Simples

  • String: "Hola mundo"
  • Number: 42, 3.14
  • Boolean: true, false

Complejos

  • Array: [1, 2, 3]
  • Object: {nombre: "Ana"}

Operadores Lógicos

&& (AND)

Verdadero si AMBAS condiciones son verdaderas

edad >= 18 && tieneID

|| (OR)

Verdadero si AL MENOS UNA es verdadera

esVIP || pagoExtra

! (NOT)

Invierte el valor booleano

!estaLloviendo
