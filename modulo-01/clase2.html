Outcome

📚 Módulo 3: JavaScript Fundamentos

Outcome

📚 Módulo 4: JavaScript Avanzado

Proyecto: App web compleja e interactiva

📚 Módulo 5: Bases de Datos

Operarás bases de datos relacionales usando SQL para obtener, manipular y definir datos.

Outcome

Proyecto: Diseño y gestión de BD

📚 Módulo 6: Node.js + Express

Desarrollarás aplicaciones web usando Node.js y Express para crear backends.

Outcome

Proyecto: Servidor web funcional

📚 Módulo 7: Acceso a Datos

Desarrollarás operaciones de consulta y manipulación de datos en Node.js con ORM.

Outcome

Proyecto: App con conexión a BD

Implementarás aplicaciones empresariales que disponibilicen servicios REST con Express.

Outcome

Proyecto: API REST completa

Construirás un plan de búsqueda laboral y te prepararás para entrevistas técnicas.

🏕️ ¿Qué es un Bootcamp?

  • Programa intensivo de formación práctica
  • Diseñado para principiantes y quienes quieren cambiar de carrera
  • Combinación de teoría + práctica
  • Enfoque en proyectos reales
  • Comunidad de aprendizaje

📅 Tu Día a Día

Mañana

4 horas de clase sincrónica en vivo por Meet

Tarde/Noche

2 horas de estudio asincrónico en la plataforma

De lunes a viernes, durante 3.5 meses.

Total: 6 horas diarias dedicadas al aprendizaje

😰 Miedos Comunes

❌ "No tengo experiencia"

✓ El bootcamp está diseñado para principiantes

❌ "Es muy rápido"

✓ Divide el aprendizaje en tareas pequeñas

❌ "Es muy difícil"

✓ La práctica construye grandes profesionales

❌ "¿Conseguiré trabajo?"

✓ Tu portafolio es tu mejor carta de presentación
