AE3 · CSS — Estilos y Responsividad

🎨 Tipos de estilos

🎯 Selectores y Box Model

/* Por etiqueta, clase, ID */

.titulo { font-size: 2rem; }
#logo { width: 100px; }

/* Box Model */
.caja {
  margin: 20px; /* espacio exterior */
  padding: 10px; /* espacio interior */
  border: 1px solid #000;
}
Bloque 2 · AE3
AE3 · Responsividad y Media Queries

📱 Mobile First

Diseñar primero para pantallas pequeñas y luego agregar estilos para pantallas más grandes con min-width.

/* Base: móvil */
.contenedor { width: 100%; }

/* Tablet → 768px o más */
@media (min-width: 768px) {
  .contenedor { width: 750px; }
}

/* Desktop → 1024px o más */
@media (min-width: 1024px) {
  .contenedor { width: 960px; }
}

🖥️ Breakpoints comunes

📱 Móvil — menos de 576px
📟 Tablet — 576px a 768px
💻 Laptop — 768px a 1024px
🖥️ Desktop — más de 1024px

💡 ¿Qué es un Plugin?

Extensión que agrega funcionalidad predefinida. Bootstrap incluye plugins JS para:

  • Carrusel de imágenes (carousel)
  • Modal / ventana emergente (modal)
  • Menú colapsable (collapse)
  • Tooltips y Popovers
  • Acordeón (accordion)

init

git init

Inicializar un repositorio local en la carpeta del proyecto.

add

Bloque 4 · AE7
AE7 · Ramas y GitHub

🌿 Trabajo con ramas

# Ver ramas existentes
git branch

# Crear y cambiar a nueva rama
git checkout -b feature/nueva

# Fusionar rama en main
git checkout main
git merge feature/nueva

# Resolver conflictos: editar archivo,
# luego git add . y git commit

🐙 GitHub

  • Repositorio remoto: copia del repo en la nube
  • git clone URL: clonar un repositorio
  • git fetch: descargar sin fusionar
  • Pull Request: propuesta de cambios para revisión
  • README.md: documentación en Markdown
  • Flujo sugerido: branch → commit → push → PR → merge
Bloque 4 · AE7
AE7 · Comandos Git Esenciales

📋 Configuración y estado

# Configurar identidad (primera vez)
