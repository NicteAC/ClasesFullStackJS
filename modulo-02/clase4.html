Bloque 1 · Variables

Variables en JavaScript

var

Forma tradicional. Alcance de función. Ya no se recomienda en código moderno.

var edad = 25;

let

Alcance de bloque. Puede cambiar su valor. Recomendada para variables que mutan.

let nombre = "Juan";

const

Alcance de bloque. Su valor no puede reasignarse. Preferida siempre que sea posible.

const PI = 3.1416;
Regla de oro: usa const por defecto. Cambia a let solo si necesitas reasignar el valor.
Bloque 1 · Variables

Declarar · Asignar · Inicializar

// Declaración (sin valor)
let nombreCompleto;

// Asignación posterior
nombreCompleto = "María García";

// Inicialización (declarar + asignar en un paso)
let edad = 30;
const PI = 3.1416;

// Usar variables en operaciones
let x = 10;
let y = 5;
let suma = x + y;
console.log(suma); // 15
Bloque 1 · Funciones Nativas

Funciones Nativas Esenciales

🖨️ console.log()

Imprime en la consola del navegador. Tu mejor aliado para depurar.

console.log("Hola mundo");

💬 alert() / confirm()

Muestra ventanas emergentes. alert informa, confirm pide confirmación.

alert("¡Hola!");
confirm("¿Seguro?");

⌨️ prompt()

Solicita entrada al usuario. Devuelve el texto ingresado como string.

let n = prompt("Tu nombre:");

🔢 parseInt() / parseFloat()

Convierten texto a número entero o decimal.

parseInt("10"); // 10
parseFloat("3.14"); // 3.14
Bloque 1 · Strings

Concatenación y Template Literals

➕ Concatenación clásica

let nombre = "Juan";
let apellido = "Pérez";
let completo = nombre + " " + apellido;
console.log(completo); // Juan Pérez

✨ Template literals (ES6+)

let nombre = "Juan";
let edad = 30;
let msg = `Hola, ${nombre}! Tenés ${edad} años.`;
console.log(msg);
Template literals usan comillas graves ` ` y permiten incrustar expresiones con ${ }. Siempre preferirlos.

Bloque 2 · Aritméticos

Operadores Aritméticos

let x = 5, y = 3;

let suma = x + y; // 8
let resta = x - y; // 2
let multiplicacion = x * y; // 15
let division = x / y; // 1.666...
let resto = x % y; // 2 (módulo)
let exponente = x ** y; // 125 (5³)

// Incremento y decremento
x++; // x = 6 (equivale a x = x + 1)
x--; // x = 5 (equivale a x = x - 1)
Bloque 2 · Asignación

Operadores de Asignación Compuestos

let z = 10;

z += 5; // z = z + 5 → 15
z -= 3; // z = z - 3 → 12
z *= 2; // z = z * 2 → 24
z /= 4; // z = z / 4 → 6
z %= 2; // z = z % 2 → 0
Los operadores compuestos hacen el código más conciso y legible. Son equivalentes a escribir la operación completa.
Bloque 2 · Comparación

Operadores de Comparación

== vs ===

let a = 5, b = "5";
a == b // true (igual valor)
a === b // false (distinto tipo)

!= vs !==

let x = 5, y = 10;
x != y // true
x !== y // true (valor y tipo)

Mayor / Menor

x > y // false
x < y // true
x >= y // false
x <= y // true

⚠️ Regla importante

Siempre usa === en lugar de ==. El triple igual compara valor Y tipo, evitando comportamientos inesperados.

Bloque 2 · Lógicos

Operadores Lógicos

AND  &&

Devuelve true solo si ambas condiciones son verdaderas.

5 > 0 && 10 > 0 // true
5 > 0 && 10 < 0 // false

OR  ||

Devuelve true si al menos una condición es verdadera.

5 > 0 || 10 < 0 // true
5 < 0 || 10 < 0 // false

NOT  !

Invierte el valor booleano. true → false y viceversa.

!(5 > 0) // false
!(10 < 0) // true

Bloque 3 · Condicionales

Sentencia if · else if · else

let hora = 13;

if (hora < 12) {
  console.log("Buenos días");
} else if (hora < 18) {
  console.log("Buenas tardes");
} else {
  console.log("Buenas noches");
}
La condición dentro de if ( ) debe evaluarse a true o false. Solo se ejecuta el primer bloque cuya condición sea verdadera.
Bloque 3 · Condicionales + Lógicos

Combinando Condiciones

Con AND &&

let edad = 25;
let tieneLicencia = true;

if (edad >= 18 && tieneLicencia) {
  console.log("Podés conducir");
} else {
  console.log("No podés conducir");
}

Con OR ||

let esEstudiante = false;
let esEmpleado = true;

if (esEstudiante || esEmpleado) {
  console.log("Tenés descuento");
} else {
  console.log("Sin descuento");
}
Bloque 3 · Switch

Sentencia Switch

let dia = 3;

switch (dia) {
  case 1:
    console.log("Lunes"); break;
  case 2:
    console.log("Martes"); break;
  case 3:
    console.log("Miércoles"); break;
  default:
    console.log("Otro día");
}
El break es obligatorio para salir del switch. Sin él, el código sigue ejecutando los casos siguientes (fall-through).
Bloque 3 · Ternario

Operador Ternario

condición ? expresión_si_true : expresión_si_false
// if/else clásico:
let rta;
if (num1 > num2) { rta = "el 1"; } else { rta = "el 2"; }

// Mismo resultado con ternario (una sola línea):
let rta = num1 > num2 ? "el 1" : "el 2";

// Otro ejemplo práctico:
const acceso = edad >= 18 ? "Permitido" : "Denegado";

Bloque 4 · Funciones

Funciones en JavaScript

📌 Declaración clásica

function saludar(nombre, edad) {
  console.log(
    `¡Hola, ${nombre}! Tenés ${edad} años.`
  );
}
saludar("Juan", 30);

⚡ Función flecha (ES6)

// Con cuerpo
const suma = (a, b) => { return a + b; };

// Una línea (return implícito)
const resta = (a, b) => a - b;

console.log(suma(15, 20)); // 35
console.log(resta(20, 5)); // 15
Las funciones flecha => son la sintaxis moderna. Úsalas para funciones cortas y callbacks.
Bloque 4 · Scope

Scope (Ámbito de Variables)

🌍 Global

Declarada fuera de funciones. Accesible desde cualquier parte del código.

let nombre = "Juan"; // global
function saludar() {
  console.log(nombre); // ✅
}
console.log(nombre); // ✅

🏠 Local

Declarada dentro de una función. Solo existe dentro de ese bloque.

function saludar() {
  let msg = "Hola"; // local
  console.log(msg); // ✅
}
console.log(msg); // ❌ Error!
Bloque 4 · DOM

¿Qué es el DOM?

🌳 Document Object Model

Representación en memoria del HTML. JavaScript puede leer y modificar cualquier elemento en tiempo real.

🔑 Objeto document

Punto de entrada al DOM. Desde él se acceden todos los métodos para manipular la página.

console.dir(document);
console.dir(document.body);
GET

Seleccionar

Acceder a elementos existentes.

MOD

Modificar

Cambiar texto, clases o estilos.

ADD

Crear

Agregar nuevos elementos al HTML.

DEL

Eliminar

Quitar elementos del árbol DOM.

Bloque 4 · DOM · Selectores

Selectores del DOM

// Por ID (devuelve un elemento)
const elem = document.getElementById('mi-id');

// Por etiqueta (devuelve colección)
const parrafos = document.getElementsByTagName('p');

// Por clase CSS (devuelve colección)
const cards = document.getElementsByClassName('card');

// querySelector: el primero que coincida (CSS selector)
const primero = document.querySelector('#mi-id .clase1');

// querySelectorAll: todos los que coincidan
const todos = document.querySelectorAll('ul li');
todos.forEach(el => console.log(el.textContent));
Preferí siempre querySelector / querySelectorAll: aceptan cualquier selector CSS y son los más modernos.
Bloque 4 · DOM · Manipulación

Modificar · Crear · Eliminar

// Modificar texto
elem.innerText = 'Nuevo texto';
elem.innerHTML = '<strong>Negrita</strong>';

// Cambiar clases CSS
elem.className = 'clase-nueva'; // reemplaza todas
elem.className += ' otra-clase'; // agrega una más

// Crear y agregar un elemento
const p = document.createElement('p');
p.textContent = 'Nuevo párrafo';
document.getElementById('contenedor').appendChild(p);

// Eliminar un elemento
document.getElementById('mi-nodo').remove();

Obtener Datos de Formularios

Bloque 4 · Eventos

Eventos en JavaScript

🖱️ Mouse

  • click → clic simple
  • dblclick → doble clic
  • mouseover → pasa el cursor
  • mouseout → sale el cursor
  • mousemove → mueve el cursor

⌨️ Teclado y Formulario

  • keydown → tecla presionada
  • keyup → tecla liberada
  • input → cambio en tiempo real
  • change → cambio confirmado
  • submit → envío de formulario
Bloque 4 · Eventos · addEventListener

addEventListener

// Sintaxis: elemento.addEventListener('evento', función)

const boton = document.getElementById('mi-boton');

// Función anónima
boton.addEventListener('click', function() {
  console.log('¡Hola, mundo!');
});

// Función flecha (más moderno)
boton.addEventListener('dblclick', () => {
  console.log('¡Doble clic!');
});

// Evento submit con preventDefault
form.addEventListener('submit', (event) => {
  event.preventDefault(); // evitar recarga
  const val = document.getElementById('name').value;
  console.log(`Enviado: ${val}`);
});
