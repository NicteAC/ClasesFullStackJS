Introducción a Bootstrap

Frameworks, instalación y sistema de grillas

Bloque 1 · Framework

¿Qué es un Framework?

📦 Definición

Marco de trabajo que trae un conjunto de librerías y herramientas para que el desarrollo de una aplicación sea más rápido y consistente.

💡 Dato clave

Un framework puede estar conformado por una o varias librerías. Bootstrap actúa como framework de CSS, y hay muchos otros disponibles en el mercado.

Ejemplos: Django (Python) · Symfony (PHP) · Spring (Java) · Bootstrap (CSS/JS)

Bloque 1 · Bootstrap

¿Qué es Bootstrap?

🌐 Definición

Framework de código abierto para diseño de sitios y aplicaciones web responsivos. Es uno de los más populares del mundo del desarrollo front-end.

📁 ¿Qué incluye?

  • Archivos CSS predefinidos
  • Archivos JavaScript para interactividad
  • Componentes listos para usar
  • Sistema de grillas responsive
Botones · Navbars · Formularios · Modales · Carruseles · Alertas — todo pre-estilizado.
Bloque 1 · Características

¿Por qué usar Bootstrap?

📱

Responsividad

Grilla flexible que se adapta automáticamente a cualquier tamaño de pantalla.

🧩

Componentes

Amplia gama de elementos pre-estilizados listos para usar. Ahorro de tiempo y esfuerzo.

🎨

Personalización

Modificar estilo y comportamiento de cualquier componente mediante clases CSS.

🌍

Compatibilidad

Diseñado para todos los navegadores modernos principales.

📖

Documentación

Documentación detallada y exhaustiva en getbootstrap.com

Bloque 1 · Incorporar Bootstrap

Incorporar Bootstrap vía CDN

<!-- 1. En el <head>: CSS de Bootstrap -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">

<!-- 2. Al final del <body>: JS de Bootstrap -->
<script
  src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js">
</script>
El CSS va en el <head> — el JS va antes de cerrar el </body>
Bloque 1 · Containers

Tipos de Contenedores

📦 .container

Ancho fijo y centrado en la página. Se adapta a diferentes breakpoints automáticamente.

<div class="container">...</div>

🌊 .container-fluid

100% del ancho de la ventana en todos los tamaños de pantalla.

<div class="container-fluid">...</div>

🖥️ .container-xl

Fluido hasta el breakpoint XL, luego ancho fijo.

📺 .container-xxl

Fluido hasta el breakpoint XXL, luego ancho fijo.

Bloque 1 · Sistema de Grillas

El Sistema de Grillas de Bootstrap

🔢 Base: 12 columnas

Bootstrap usa una rejilla flexible de 12 columnas. Puedes combinarlas libremente: 6+6, 4+4+4, 3+9, etc.

🏗️ Estructura

  • .row → crea una fila
  • .col-{size}-{n} → columna de n/12
  • Las columnas van dentro de filas
  • Las filas van dentro de contenedores
xs

Extra small

< 576px

sm

Small

≥ 576px

md

Medium

≥ 768px

lg

Large

≥ 992px

xl

X-Large

≥ 1200px

xxl

XX-Large

≥ 1400px

Bloque 1 · Grillas · Ejemplo

Grilla en Práctica

<div class="container">
  <div class="row">
    <div class="col-sm-6"> ← ocupa 6/12 columnas en sm+
      Columna izquierda
    </div>
    <div class="col-sm-6"> ← ocupa 6/12 columnas en sm+
      Columna derecha
    </div>
  </div>
</div>
En pantallas pequeñas las columnas se apilan verticalmente. En sm+ se muestran lado a lado.
Bloque 2 · Tablas

Tablas en Bootstrap

📊 Clase base

Agrega .table a cualquier <table> para aplicar estilos Bootstrap.

<table class="table">

🎨 Variantes de color

  • .table-dark → fondo oscuro
  • .table-striped → filas alternas
  • .table-hover → highlight al pasar
  • .table-bordered → con bordes

🔦 Filas y celdas

Clases en <tr> o <td>: .table-primary, .table-danger, .table-success…

📲 Tabla responsiva

Envuelve la tabla en .table-responsive para permitir scroll horizontal en móviles.

Bloque 3 · Hero

Encabezado Destacado (Hero)

🦸 ¿Qué es?

Sección principal llamativa que ocupa el ancho de la pantalla. Ideal para mensajes principales, introducciones o llamadas a la acción.

🔧 En Bootstrap 5

Se construye combinando .container con clases de fondo, espaciado y tipografía. No requiere componente especial.

<div class="container bg-light p-5 rounded-3">
  <h1 class="display-5">Título Principal</h1>
  <p class="lead">Descripción del sitio...</p>
  <button class="btn btn-primary">Acción</button>
</div>
Bloque 3 · Hero full-width

Hero a Pantalla Completa

<!-- container-fluid para que ocupe todo el ancho -->
<div class="container-fluid bg-primary text-white py-5 text-center">
  <h1 class="display-4">Bienvenido a mi sitio</h1>
  <p class="lead">Una descripción breve y atractiva</p>
  <a href="#" class="btn btn-light btn-lg">Comenzar</a>
</div>
Combinando .bg-* + .text-* + .py-* obtienes secciones visuales atractivas sin CSS extra.
Bloque 3 · Alertas

Alertas en Bootstrap

📢 Estructura básica

Clase .alert + variante de color en un <div>.

🔵 .alert-primary
🟢 .alert-success
🔴 .alert-danger
🟡 .alert-warning

⚙️ Variantes

  • .alert-heading → título dentro de la alerta
  • .alert-link → enlace estilizado
  • .alert-dismissible → con botón de cierre
  • .fade .show → animación de aparición
  • data-bs-dismiss="alert" → cerrar con JS
Bloque 3 · Alertas · Código

Alerta con Cierre y Animación

<!-- Alerta descartable con animación -->
<div class="alert alert-warning alert-dismissible fade show" role="alert">
  <h4 class="alert-heading">¡Atención!</h4>
  <p>Este mensaje es importante para el usuario.</p>
  <a href="#" class="alert-link">Ver más</a>
  <button type="button" class="btn-close"
    data-bs-dismiss="alert" aria-label="Cerrar"></button>
</div>

Bloque 4 · Formularios

Formularios en Bootstrap

✏️ Campos de texto

Clase .form-control en <input> y <textarea> para estilos Bootstrap responsivos.

📋 Selección

<select> con .form-control o .form-select para listas desplegables estilizadas.

🔘 Agrupación

.form-group agrupa campos relacionados. .form-check para checkboxes y radios.

🚀 Botón envío

Usar <button type="submit"> con .btn .btn-primary para el botón de envío del formulario.

Bloque 4 · Formularios · Código

Formulario de Contacto

<form>
  <div class="mb-3">
    <label for="nombre">Nombre:</label>
    <input type="text" class="form-control" id="nombre" required>
  </div>
  <div class="mb-3">
    <label for="email">Email:</label>
    <input type="email" class="form-control" id="email" required>
  </div>
  <div class="mb-3">
    <textarea class="form-control" placeholder="Tu mensaje..."></textarea>
  </div>
  <button type="submit" class="btn btn-primary">Enviar</button>
</form>
Bloque 4 · Formularios · Validación

Validación de Formularios

✅ Estados válidos

Clase .is-valid en el input + .valid-feedback para el mensaje de confirmación.

❌ Estados inválidos

Clase .is-invalid en el input + .invalid-feedback para el mensaje de error.

🔤 HTML5 nativo

Combinar atributos HTML5 como required, type="email" con las clases de Bootstrap.

📦 Input groups

.input-group permite agrupar inputs con texto o botones adyacentes de forma elegante.

Bloque 4 · Navbar

Barra de Navegación (Navbar)

🧭 Estructura básica

Elemento <nav> con clase .navbar. Dentro: brand, toggler y collapse con los enlaces.

📱 Colapso responsivo

.navbar-toggler muestra el botón hamburguesa en móviles. .navbar-collapse envuelve los items.

🎨 Estilos

  • .navbar-light → fondo claro
  • .navbar-dark → fondo oscuro
  • .bg-light/.bg-dark → color de fondo
  • .navbar-expand-lg → breakpoint

🔗 Items

  • .navbar-brand → logo / nombre
  • .navbar-nav → lista de items
  • .nav-item → cada elemento
  • .nav-link → enlace estilizado
Bloque 4 · Navbar · Código

Navbar Responsiva Completa

<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <a class="navbar-brand" href="#">MiSitio</a>
  <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#menu">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="menu">
    <ul class="navbar-nav ms-auto">
      <li class="nav-item"><a class="nav-link active" href="#">Inicio</a></li>
      <li class="nav-item"><a class="nav-link" href="#">Servicios</a></li>
      <li class="nav-item"><a class="nav-link" href="#">Contacto</a></li>
    </ul>
  </div>
</nav>
