🌐 Bloque 1

¿Qué es el Desarrollo Web?

Implica crear y mantener sitios web y aplicaciones en Internet mediante tecnologías y lenguajes de programación.

🎨 Front-End

Se enfoca en la parte visual y la experiencia del usuario. Usa HTML, CSS y JavaScript.

⚙️ Back-End

Gestiona la lógica del servidor y los datos. Usa PHP, Python, Ruby, Java.

Esta especialización garantiza productos de alta calidad y mantenimiento eficiente.

⚡ Diferencias Clave

Front-End · Back-End · Fullstack

🎨

Front-End

  • Interfaz visible
  • HTML, CSS, JS
  • Usabilidad y accesibilidad
  • Responsive design
⚙️

Back-End

  • Lógica del servidor
  • PHP, Python, Java
  • Bases de datos
  • Seguridad y rendimiento
🌟

Fullstack

  • Domina ambos lados
  • Visión completa
  • Trabajo en equipo
  • Alta demanda laboral
🌍 Internet y Estándares

El Navegador y la W3C

🌐 Internet & Navegador

  • Red global de computadoras conectadas
  • Protocolo TCP/IP para comunicación
  • Barra de dirección → URLs
  • Interpreta HTML, CSS y JS

📋 W3C (desde 1994)

  • Fundada por Tim Berners-Lee
  • Desarrolla estándares web abiertos
  • Promueve accesibilidad y privacidad
  • HTML · CSS · HTTP · SVG · WCAG

HTML evolucionó: HTML 1 (1991) → HTML5 (2004-2008) → Estándar actual con semántica, multimedia y APIs nativas.

🔺 La Triada del Desarrollo Web

HTML + CSS + JavaScript

🏗️

HTML

Define la estructura y contenido. Etiquetas, párrafos, imágenes, enlaces.

🎨

CSS

Controla la presentación: colores, fuentes, márgenes, layouts y animaciones.

⚡

JavaScript

Agrega interactividad: eventos, animaciones, validaciones y comunicación con el servidor.

Analogía: HTML = esqueleto · CSS = ropa · JS = movimiento

🏷️ Bloque 4 — HTML5

¿Qué es HTML5?

HyperText Markup Language — Lenguaje principal para crear la estructura y contenido de páginas web. Usa etiquetas para definir elementos e indica al navegador cómo presentar la información.

🔧 Elementos

Definidos con etiquetas de apertura y cierre que delimitan el contenido.

🏷️ Etiquetas

Palabras clave entre < y > que marcan el inicio y fin de un elemento.

<etiqueta atributo="valor"> Contenido </etiqueta>
📄 Estructura Básica

Esqueleto de un Documento HTML

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Mi sitio web</title>
  </head>
  <body>
    <!-- Contenido visible aquí -->
  </body>
</html>

🧠 <head>

Información para el navegador: título, charset, estilos, metadatos.

👁️ <body>

Todo lo visible para el usuario: textos, imágenes, formularios.

🧩 Semántica HTML

Etiquetas Semánticas

<header> — Cabecera de la página
<nav> — Navegación principal
<main> — Contenido principal
<section> — Sección temática
<article> — Artículo independiente
<aside> — Contenido lateral
<footer> — Pie de página
<figure> — Imagen con descripción

✅ Mejoran el SEO, la accesibilidad y la legibilidad del código.

📝 Contenido HTML

Encabezados · Párrafos · Listas

📌 Encabezados

Jerarquía <h1> a <h6>. Solo un <h1> por documento.

<h1>Principal</h1>
<h2>Secundario</h2>
<h3>Terciario</h3>

📄 Párrafo

La etiqueta <p> agrupa bloques de texto continuo.

<p>Lorem ipsum...</p>

📋 Listas

<ul> desordenada · <ol> ordenada · <li> ítem.

<ul>
 <li>Item 1</li>
</ul>
🔗 Bloque 5 — Links & Imágenes

Enlaces e Hipervínculos

🌐 Ruta Absoluta

<a href="https://www.google.cl"
   target="_blank">
  Google
</a>

Dirección completa con protocolo y dominio.

📁 Ruta Relativa + Imagen

<a href="./pagina2.html">Ir</a>

<img src="imagen.jpg"
     alt="Descripción">

Relativa al archivo actual. alt mejora accesibilidad.

target="_blank" abre en nueva pestaña · target="_self" abre en la misma (por defecto).

📊 Tablas HTML

Creación de Tablas

🏷️ Etiquetas

  • <table> — Define la tabla
  • <tr> — Fila (table row)
  • <th> — Celda de encabezado
  • <td> — Celda de datos
  • <thead> / <tbody> — Secciones

💻 Ejemplo

<table>
 <tr>
  <th>País</th> <th>Habitantes</th>
 </tr>
 <tr>
  <td>China</td> <td>1.4B</td>
 </tr>
</table>
📝 Formularios HTML

Creación de Formularios

🔧 Atributos <form>

  • action — URL destino de los datos
  • method="GET" — datos en la URL
  • method="POST" — datos ocultos
  • <label> — Etiqueta descriptiva
  • required — Campo obligatorio

💻 Ejemplo Login

<form method="POST">
 <label>Usuario:</label>
 <input type="text" required>
 <label>Contraseña:</label>
 <input type="password" required>
 <input type="submit" value="Entrar">
</form>
🧱 Contenedores

<div> y <span>

📦 <div> — Bloque

  • Contenedor de bloque
  • Ocupa todo el ancho disponible
  • Ideal para secciones y layouts
  • Base para maquetación web
<div class="header">
  <h1>Título</h1>
</div>

✂️ <span> — Inline

  • Contenedor en línea
  • No rompe el flujo del texto
  • Aplica estilos a partes del texto
  • No afecta la estructura
<p>Texto con <span>énfasis</span> aquí.</p>
