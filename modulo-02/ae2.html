¿Qué es HTML?

HTML es el lenguaje fundamental para estructurar contenido web

HyperText Markup Language

  • Lenguaje de marcado, no de programación
  • Define la estructura y semántica del contenido
  • Base de toda página web

HTML5 (Versión Actual)

  • Elementos semánticos mejorados
  • Mejor soporte multimedia
  • APIs modernas integradas
  • Mayor accesibilidad y SEO

Elementos, Etiquetas y Atributos

Etiquetas

Símbolos que marcan y delimitan elementos

Elementos

Conjunto completo: apertura + contenido + cierre

Atributos

Información adicional sobre un elemento

Contenido
</etiqueta>

Los atributos configuran propiedades específicas

Estructura Básica de un Documento HTML

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>Mi sitio web</title> </head> <body> <!-- Contenido visible va aquí --> </body> </html>

<head>

Información para el navegador: título, codificación, estilos

<body>

Todo lo visible para el usuario

Encabezados (h1 - h6)

Jerarquía Semántica

  • <h1>: Título principal (solo uno por página)
  • <h2>: Secciones principales
  • <h3>-<h6>: Subsecciones
  • Mantener orden lógico
  • No saltar niveles
<h1>Reino Animal</h1> <h2>Vertebrados</h2> <h3>Mamíferos</h3> <h4>Bípedos</h4> <h4>Cuadrúpedos</h4> <h3>Aves</h3> <h2>Invertebrados</h2>

Párrafos y Listas

Párrafos <p>

Bloques de texto continuo

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>

Listas Desordenadas <ul>

<ul> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul>

Texto Destacado

  • <strong>: Negrita importante
  • <em>: Énfasis (cursiva)

Listas Ordenadas <ol>

<ol> <li>Primer paso</li> <li>Segundo paso</li> <li>Tercer paso</li> </ol>

Enlaces e Hipervínculos

URLs y Rutas

Rutas Absolutas

URL completa con protocolo y dominio

<a href="https://www.ejemplo.com/pagina"> Enlace </a>

https:// = protocolo
www.ejemplo.com = dominio
/pagina = ruta

Rutas Relativas

Relativas al archivo actual

./archivo.html ../carpeta/otro.html ../imagenes/foto.jpg

./ = directorio actual
../ = subir un nivel

Atributos de Enlaces

Atributo href

"Hipertext Reference" - define el destino

<a href="https://google.cl"> Google </a> <a href="pagina2.html"> Página 2 </a>

Atributo target

Controla cómo se abre el enlace

  • _blank: Nueva pestaña
  • _self: Misma ventana (default)
  • _parent: Marco principal
  • _top: Ventana completa
<a href="..." target="_blank"> Abrir en nueva pestaña </a>

Imágenes en HTML

Etiqueta <img>

Elemento de autocierre (no necesita </img>)

<img src="ruta/imagen.jpg" alt="Descripción">

Atributos esenciales:

  • src: Ruta de la imagen
  • alt: Texto alternativo (accesibilidad)

Ejemplos

<!-- Imagen externa --> <img src="https://ejemplo.com/foto.jpg" alt="Paisaje"> <!-- Imagen local --> <img src="./imagenes/logo.png" alt="Logo empresa"> <!-- Con dimensiones --> <img src="foto.jpg" alt="Foto" width="500">

Tablas en HTML

Etiquetas Principales

  • <table>: Define la tabla
  • <tr>: Fila (table row)
  • <th>: Celda de encabezado
  • <td>: Celda de datos

Estructura Opcional

  • <thead>: Cabecera de tabla
  • <tbody>: Cuerpo de tabla
  • <tfoot>: Pie de tabla
<table> <tr> <th>Nombre</th> <th>Edad</th> <th>Ciudad</th> </tr> <tr> <td>Ana</td> <td>25</td> <td>Santiago</td> </tr> <tr> <td>Carlos</td> <td>30</td> <td>Valparaíso</td> </tr> </table>

Elementos de Estructura: DIV y SPAN

<div> - Division

Contenedor de bloque

  • Agrupa elementos en bloques
  • Ocupa todo el ancho disponible
  • Inicia en nueva línea
  • Ideal para layout y secciones
<div> <h2>Título</h2> <p>Contenido...</p> </div>

<span> - Span

Contenedor en línea

  • Agrupa texto o elementos pequeños
  • Solo ocupa el espacio necesario
  • No inicia nueva línea
  • Ideal para estilos específicos
<p>Texto con <span style="color: red"> palabra destacada </span> en rojo</p>
