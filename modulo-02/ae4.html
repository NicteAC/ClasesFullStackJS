Bootstrap Framework

Desarrollo Web Responsivo con CSS y JavaScript

¿Qué es Bootstrap?

Framework de Código Abierto

Bootstrap es el framework CSS más popular para diseñar sitios web responsivos con HTML, CSS y JavaScript

Componentes Predefinidos

Conjunto de archivos CSS, JavaScript y componentes listos para usar: botones, navegación, formularios, grillas y más

Diseño Responsivo

Sistema de grillas flexible que se adapta automáticamente a diferentes tamaños de pantalla

Compatibilidad Total

Funciona consistentemente en todos los navegadores modernos

Beneficios de usar Bootstrap

  • Ahorro de tiempo: Componentes prediseñados y probados
  • Responsividad garantizada: Mobile-first por defecto
  • Consistencia visual: Sistema de diseño cohesivo
  • Personalización sencilla: Clases y variables CSS
  • Comunidad activa: Documentación extensa y soporte
  • Compatibilidad cross-browser: Menos problemas de CSS

Sistema de Grillas de Bootstrap

Grilla de 12 Columnas

Bootstrap usa un sistema flexible basado en 12 columnas que se adapta a cualquier dispositivo

Breakpoints

  • sm - Pequeño (≥576px)
  • md - Mediano (≥768px)
  • lg - Grande (≥992px)
  • xl - Extra grande (≥1200px)

Clases de Columnas

.col-{breakpoint}-{número}

.col-12      → 100% ancho
.col-md-6    → 50% en md+
.col-lg-4    → 33% en lg+

Contenedores y Tablas

Tipos de Contenedores

  • .container - Ancho fijo centrado
  • .container-fluid - Ancho 100%
<div class="container">
  <div class="row">
    <div class="col-md-6">...</div>
  </div>
</div>

Tablas con Bootstrap

  • .table - Estilo básico
  • .table-striped - Filas alternadas
  • .table-hover - Efecto hover
  • .table-responsive - Scroll horizontal

Utilidades de Espaciado

Margin y Padding

  • m-{tamaño} - Margin en todos los lados
  • mt-3 - Margin top
  • mb-4 - Margin bottom
  • p-5 - Padding grande
  • py-3 - Padding vertical
  • mx-auto - Centrar horizontalmente

Tamaños

  • 0 - 0px
  • 1 - 0.25rem (4px)
  • 2 - 0.5rem (8px)
  • 3 - 1rem (16px)
  • 4 - 1.5rem (24px)
  • 5 - 3rem (48px)

Formularios en Bootstrap

Campos Básicos

<div class="mb-3">
  <label class="form-label">
    Email
  </label>
  <input type="email" 
         class="form-control" 
         required>
</div>

Validación

<form class="needs-validation" 
      novalidate>
  <input type="text" 
         class="form-control" 
         required>
  <div class="invalid-feedback">
    Campo requerido
  </div>
</form>

Barra de Navegación

Navbar Responsivo

<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Logo</a>
    
    <button class="navbar-toggler" data-bs-toggle="collapse" 
            data-bs-target="#menu">
      <span class="navbar-toggler-icon"></span>
    </button>
    
    <div class="collapse navbar-collapse" id="menu">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link" href="#">Inicio</a>
        </li>
      </ul>
    </div>
  </div>
</nav>
