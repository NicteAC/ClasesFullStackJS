📱 Actividad Práctica: Media Queries

⏱️ Tiempo: 20 minutos

🎯 Contexto

Eres un diseñador web contratado por una empresa de tecnología financiera para crear una página web de su wallet digital. La wallet permite a los usuarios administrar sus activos digitales y realizar transacciones de criptomonedas.

📋 Consigna

Diseñar una página web responsiva que ofrezca una experiencia óptima en dispositivos móviles, tablets y computadoras de escritorio.

1

Estructura HTML

Crea un archivo HTML llamado wallet.html y define la estructura básica de la página web.

Incluye:

  • Un encabezado con el nombre de la wallet
  • Un menú de navegación
  • Un área de resumen de cuentas
  • Un área para realizar transacciones
  • Un pie de página con información de contacto
2

Creación de Estilos CSS

Crea un archivo CSS llamado styles.css para definir los estilos de la página web.

Comienza con estilos generales que se aplicarán a todas las pantallas. Estos estilos deben incluir:

  • La fuente
  • Los colores
  • Los márgenes básicos
3

Media Queries

Utiliza media queries en tu archivo CSS para adaptar el diseño a diferentes tamaños de pantalla:

  • 📱 Dispositivos móviles: Ancho máximo de 767px
  • 💻 Tablets: Ancho mínimo de 768px y ancho máximo de 1023px
  • 🖥️ Computadoras de escritorio: Ancho mínimo de 1024px

Dentro de cada media query, ajusta los estilos para que el contenido y la disposición sean adecuados para cada tamaño de pantalla. Presta especial atención a:

  • Los elementos de navegación
  • La disposición de la información de la cuenta
  • Las funcionalidades de transacción
4

Diseño Responsivo

Asegúrate de que el diseño sea responsivo y se vea bien en cada una de las tres categorías de tamaños de pantalla.

Esto incluye:

  • El reordenamiento de elementos
  • Cambios en el tamaño de fuente
  • Ajustes en los márgenes y espaciado

💡 Tip: Prueba tu diseño cambiando el tamaño de la ventana del navegador o usando las herramientas de desarrollador (F12) para simular diferentes dispositivos.

💻 Código de Ejemplo

Puedes copiar este código como base para tu proyecto

📄 wallet.html 
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="stylesheet" href="styles.css" />
    <title>Wallet Digital</title>
  </head>
  <body>
    <header>
      <h1>My Wallet</h1>
      <nav>
        <ul>
          <li><a href="#">Inicio</a></li>
          <li><a href="#">Cuentas</a></li>
          <li><a href="#">Transacciones</a></li>
          <li><a href="#">Configuración</a></li>
        </ul>
      </nav>
    </header>
    <main>
      <section class="account-summary">
        <h2>Resumen de la Cuenta</h2>
        <!-- Contenido del resumen de la cuenta -->
      </section>
      <section class="transactions">
        <h2>Transacciones</h2>
        <!-- Contenido de las transacciones -->
      </section>
    </main>
    <footer>
      <p>Contacto: info@example.com</p>
    </footer>
  </body>
</html>
🎨 styles.css 
/* Estilos generales para todas las pantallas */
body {
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}

header {
  background-color: #333;
  color: white;
  text-align: center;
  padding: 10px;
}

nav ul {
  list-style-type: none;
  padding: 0;
}

nav ul li {
  display: inline;
  margin-right: 20px;
}

main {
  max-width: 800px;
  margin: 0 auto;
  padding: 20px;
}

footer {
  background-color: #333;
  color: white;
  text-align: center;
  padding: 10px;
}

/* Media Query para dispositivos móviles */
@media (max-width: 767px) {
  header {
    font-size: 20px;
  }
  nav ul {
    text-align: center;
  }
  nav ul li {
    display: block;
    margin: 10px 0;
  }
  main {
    padding: 10px;
  }
  footer {
    font-size: 14px;
  }
}

/* Media Query para tablets */
@media (min-width: 768px) and (max-width: 1023px) {
  header {
    font-size: 24px;
  }
  nav ul li {
    margin-right: 30px;
  }
  main {
    padding: 20px;
  }
  footer {
    font-size: 16px;
  }
}

/* Media Query para computadoras de escritorio */
@media (min-width: 1024px) {
  header {
    font-size: 28px;
  }
  nav ul li {
    margin-right: 40px;
  }
  main {
    padding: 30px;
  }
  footer {
    font-size: 18px;
  }
}

🚀 Cómo usar este código en VS Code:

  1. Copia el código HTML: Click en "Copiar Código" en la sección HTML
  2. Crea el archivo: En VS Code, crea un archivo llamado wallet.html
  3. Pega el código: Pega el código HTML copiado en el archivo
  4. Copia el código CSS: Cambia a la pestaña CSS y copia ese código
  5. Crea el archivo CSS: En la misma carpeta, crea styles.css
  6. Pega el código CSS: Pega el código CSS en el archivo
  7. Abre en navegador: Click derecho en wallet.html → "Open with Live Server" o abre directamente en tu navegador
  8. Prueba el responsive:
    • Cambia el tamaño de la ventana del navegador
    • Presiona F12 para abrir DevTools
    • Click en el ícono de dispositivo móvil (o Ctrl+Shift+M)
    • Prueba diferentes tamaños: móvil, tablet, desktop
